# Kagi Assistant Desktop

A desktop application for [Kagi Assistant](https://kagi.com/assistant) built with Tauri.

## Build

### Prerequisites

- [Rust](https://www.rust-lang.org/)
- [Bun](https://bun.sh/)

### Building

```bash
bun install

bun run tauri build
```

This will build to `src-tauri/target/release/bundle`.
